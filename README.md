Created by: Evan L and Neil K

This is a fun F1 themed obstacle dodging game.
It starts with a login with your account.
Then you customize your car and number.
Now it is a real F1 reaction test start.
And then you dodge other cars to get points.
Use arrow keys and WASD to move around the screen.
